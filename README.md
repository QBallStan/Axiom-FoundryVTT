# Axiom//Core FoundryVTT System

This is the first development release of the official **Axiom//Core** system for **FoundryVTT V14**.

It is still an early testing version, so some features are incomplete and breaking changes should be expected.

## What’s included

This release includes support for **Protagonist** and **NPC** actors, along with the main item types needed for early play:

- **Skills**
- **Traits**
- **Equipment**
- **Armor**
- **Melee Weapons**
- **Ranged Weapons**
- **Ammunition**

The system includes basic character and NPC sheets, item sheets, wound tracking, Fate, Momentum, Action Points, currency settings, Active Effects support, and early token HUD/status integration.

## Automation

This version includes the first pass of automation for:

- **d100 roll-under tests**
- **Success Targets**
- **Hits**
- **Outcome tiers**
- **Complications**
- **Roll modifiers**
- **Timeframes**
- **Initiative**
- **Action Points**
- **Momentum**
- **Melee attacks**
- **Ranged attacks**
- **Ammunition use**
- **Armor**
- **Toughness**
- **Hit locations**
- **Damage**
- **Wounds**

Combat automation is functional enough for early testing, but still incomplete.
